# Sutlej River Water Quality Prediction using Machine Learning

## 📌 Overview
This project predicts Water Quality Index (WQI) using Random Forest Machine Learning model based on physicochemical parameters.

## 📊 Parameters Used
- DO (Dissolved Oxygen)
- BOD (Biological Oxygen Demand)
- COD (Chemical Oxygen Demand)
- pH
- TDS
- Turbidity
- Nitrates
- Temperature
- etc.

## 🧮 Methodology
- Data preprocessing
- WQI calculation (Brown’s Method)
- Random Forest Regression
- Model evaluation (MAE, R² Score)

## 📈 Results
- R² Score: ~0.96
- High prediction accuracy

- Real-time monitoring system
- Integration with IoT sensors
